Atividades:


1. criar uma tela de desenho sentível ao toque (i.e. tocar na tela branca e desenhar)
2. criar tela que captura uma imagem da galeria
3. aplicar filtro na imagem obtida
4. desenhar a imagem pós-filtro na tela de desenho
